### Scope and Structure

- Mathematical formulation of both DoA estimation methods  
- Python-based simulations under interference and additive white Gaussian noise  
- Real-world measurements using a linear equidistant array of four log-periodic antennas

- Comparative evaluation of the Fourier and Capon methods using simulated and measured data  
- Assessment of accuracy, noise robustness, and interference performance
